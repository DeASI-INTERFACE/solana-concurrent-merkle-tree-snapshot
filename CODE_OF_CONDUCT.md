# Code of Conduct

## Our Standards

This repository is a professional compliance and audit tool. All contributors and participants are expected to:

- Use professional, precise language when discussing on-chain data.
- Provide citations for any on-chain claims.
- Respect the authority and governance structure of the documented on-chain accounts.
- Avoid speculative or unverified assertions about account ownership or program behavior.

## Enforcement

Instances of unprofessional conduct, fabricated data, or deliberate omission of required fields may result in removal from the contributor list. Report issues to [@De-ASI-INTERFACE](https://github.com/De-ASI-INTERFACE).
